# Sherlock Telegram Bot

Telegram бот для поиска аккаунтов по никнейму с помощью [Sherlock](https://github.com/sherlock-project/sherlock).

## Деплой на Railway

1. Форкни или склонируй этот репозиторий
2. Зайди на [railway.app](https://railway.app) и создай новый проект из GitHub
3. В разделе **Variables** добавь:
   - `BOT_TOKEN` — токен от @BotFather
   - `ADMIN_ID` — твой Telegram ID (можно узнать у @userinfobot)
4. Railway автоматически задеплоит бота

## Использование

Просто напиши никнейм в чат боту — он найдёт все аккаунты с таким именем на сотнях сайтов.
